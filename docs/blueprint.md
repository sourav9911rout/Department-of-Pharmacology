# **App Name**: Departmental Hub

## Core Features:

- Procured Items Tracking: Display a sortable, filterable table of procured items with details like name, category, quantity, and procurement date.
- Requirement List Management: Two sub-tabs for primary and secondary requirements, displaying item name, required quantity, priority, and status. Admins can update quantities and status.
- Classes & Meetings Schedule: Display upcoming and past classes/meetings, sorted by date, with topic, date, time, instructor, and a Google Meet link. Admin can add, edit, or delete entries.
- Google Meet Integration: Integrate Google Meet links for easy joining of classes and meetings.
- Daily Drug Highlight Redirect: Tab that redirects to the external pharmacology resource at https://pharmacology.vercel.app.
- Document & Resources Access: Embed or link to Google Drive folders/files for easy access to procurement documents, SOPs, and guidelines.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to inspire confidence.
- Background color: Soft, desaturated blue (#E3F2FD).
- Accent color: Muted purple (#9575CD) to subtly draw the user's attention to interactive elements.
- Body and headline font: 'PT Sans' for a modern and readable style.
- Use clear, department-friendly icons to represent each tab and data category.
- Modern, card-based layout for key information on the home dashboard. Use a sticky top navigation for easy access to tabs.
- Subtle animations and transitions for a smooth and engaging user experience.
